=============================================
Músic Player Download Youtube

Criado por tucano CMGC  um programa para 
baixar músicas do yotube em formato mp4
e converte-las em mp3 criar pastas, 
gerencia-las como listas de reprodução 
escrito em shell script.


 viva ao software livre!

==============================================
Programas necessários ( dependentes ) 


youtube-dl
	https://youtube-dl.org/

ffmpeg
	https://ffmpeg.org/

mpg123
	https://www.mpg123.org/

==========    INSTALAÇÃO   ======================

Efetue o download dos arquivos 

	player
	README.txt
	install


entra na pasta de download 

chmod +x install

./install

chmod +x player

sudo cp player /usr/bin/mpdy

==================================================
Entre na pasta de download dos arquivos

Execute o camando ( salvar o arquivo em /usr/bin/ )

copiar o arquivo player renomenado como "mpdy"
===================================================
Permissão de execução no arquivo

chmod +x player

===================================================

Mover o arquivos para odiretório dos "binários"

sudo cp player /usr/bin/mpdy

===================================================
Para desinstalar

chmod +x uninstall

./unistall
